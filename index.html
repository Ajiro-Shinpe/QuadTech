Sucii - Socializing & Ally

Succi is a dynamic social networking platform built with PHP that offers a comprehensive user experience. The platform features secure authentication and email verification, real-time user-to-user chat, follow/unfollow, like/unlike, and comment functionality. Additionally, users can customize their profiles with short descriptions, making them ideal for connecting with others.

Elegance - Salon Management System

Elegance Salon Management System is a user-friendly, responsive web application built with Laravel designed to streamline salon operations. This comprehensive platform allows clients to book appointments for various services, provide feedback, and contact the salon directly. Meanwhile, the intuitive admin panel enables easy management of appointments, services, and customer interactions.
